Peringatan Keamanan

Aplikasi ini dibuat hanya untuk tujuan pembelajaran. Jangan gunakan untuk menyimpan kata sandi asli atau data sensitif Anda.

Tentang Enkripsi XOR

Aplikasi ini menggunakan logika enkripsi XOR (Exclusive OR) yang sangat sederhana dan lemah dari segi keamanan. XOR encryption mudah untuk dipecahkan dan tidak boleh digunakan untuk melindungi data sensitif.

  • XOR encryption dapat dengan mudah dibalik jika attacker mengetahui atau menebak sebagian dari plaintext
  • Tidak ada proteksi terhadap brute-force attacks
  • Tidak ada key derivation function (KDF) yang kuat
  • Tidak ada salt atau initialization vector (IV)

Gunakan aplikasi ini hanya untuk belajar tentang konsep password manager dan enkripsi dasar. Untuk penggunaan nyata, gunakan password manager yang sudah teruji keamanannya seperti Bitwarden, 1Password, atau KeePass.
