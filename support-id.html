Pertanyaan Umum

Apa itu Aminmart Password Manager?

Aminmart Password Manager adalah aplikasi pengelola kata sandi offline yang dirancang untuk menyimpan dan mengamankan kata sandi Anda tanpa harus terhubung ke internet. Data Anda tetap berada di perangkat Anda sendiri dan tidak pernah diunggah ke server manapun.

Apakah aman menggunakan aplikasi ini?

Ya, sangat aman. Aplikasi ini menggunakan enkripsi AES-256 untuk melindungi data sensitif seperti kata sandi dan catatan. Karena bersifat offline, tidak ada risiko data Anda dicuri melalui internet. Data hanya disimpan secara lokal di perangkat Anda.

Bagaimana cara menambahkan kata sandi baru?

Buka aplikasi dan ketuk tombol "+" atau "Tambah" di layar utama. Isi detail akun seperti judul, nama pengguna, kata sandi, dan situs web. Anda juga dapat menetapkan kategori dan menambahkan catatan tambahan jika diperlukan.

Apakah saya bisa menggunakan fitur biometrik?

Ya, aplikasi ini mendukung autentikasi sidik jari dan pengenalan wajah (jika perangkat Anda mendukungnya). Anda dapat mengaktifkan fitur ini di menu pengaturan untuk membuka kunci aplikasi dengan lebih cepat dan aman.

Bagaimana jika saya lupa kata sandi utama?

Karena aplikasi ini bersifat offline dan tidak memiliki server pusat, tidak ada cara untuk memulihkan kata sandi utama. Jika Anda lupa kata sandi utama, satu-satunya solusi adalah menghapus data aplikasi dan memulai dari awal. Pastikan untuk mencadangkan data penting secara berkala.

Apakah aplikasi ini tersedia untuk iOS?

Saat ini, Aminmart Password Manager hanya tersedia untuk perangkat Android. Versi iOS sedang dalam pengembangan dan akan dirilis di masa mendatang.

Pemecahan Masalah

Aplikasi tidak bisa dibuka

  • Restart perangkat Anda dan coba buka kembali aplikasi
  • Periksa apakah aplikasi memiliki izin yang diperlukan
  • Hapus cache aplikasi melalui pengaturan perangkat
  • Jika masalah terus berlanjut, instal ulang aplikasi

Fitur biometrik tidak berfungsi

  • Pastikan fitur sidik jari/wajah aktif di pengaturan perangkat
  • Periksa apakah sensor biometrik bersih dan tidak rusak
  • Nonaktifkan dan aktifkan kembali fitur biometrik di pengaturan aplikasi
  • Pastikan aplikasi memiliki izin untuk mengakses fitur biometrik
